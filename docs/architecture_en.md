## Code Structure

```
--
 |- controllers
 |- models
 |- routers
 |- services
 |- settings
 |- utils
 |- views/static

```

controllers: REST api processor needed by Beego framework.
models: wraps HA cluster manager commands.
routers: bind URLs and controllers.
services: currently only session service.
settings: application settings.
utils: common utils.
views: static files.
